# Python学生宿舍管理系统 python678

## 项目概述

本项目为Python学生宿舍管理系统，旨在通过信息化手段提高宿舍管理的效率与准确性。系统提供宿舍分配、宿舍保修、宿舍查询等核心功能，同时包含学生信息管理、宿舍信息管理、通知管理等模块。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Flask
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 运行环境

- Python版本：3.0以上
- MySQL版本：8.0

## 功能模块

- 宿舍分配
- 宿舍保修
- 宿舍查询
- 学生信息管理
- 宿舍信息管理
- 通知管理

## 系统角色

- 管理员：负责系统管理、宿舍分配、通知发布等
- 学生：进行宿舍查询、保修申请等

## 目录结构

```
 project/
 ├── app/             # Flask应用目录
 │   ├── static/      # 静态文件目录
 │   ├── templates/   # 前端模板目录
 │   ├── main.py      # 主程序文件
 │   └── models.py    # 数据模型文件
 ├── venv/            # 虚拟环境目录
 └── requirements.txt # 项目依赖列表
```

## 部署步骤

1. 确保Python版本为3.0以上，安装MySQL 8.0数据库。
2. 创建数据库及用户，并根据项目需求导入数据表结构。
3. 安装依赖项，配置虚拟环境。
4. 修改配置文件中的数据库连接信息。
5. 运行主程序。

- 系统稳定运行，经过测试无bug。
- 代码完全开源，可运行，易于根据步骤自行配置环境。
